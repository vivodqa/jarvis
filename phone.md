# 指令名稱：phone

| 說明 | 條件數 | 來源 | 行為 |
| --- | --- | --- | --- |
| 查詢分機表 | 3 | LDAP通訊錄資料 | 針對「公司名稱, 部門, 姓名, 分機」找尋，符合所有「搜尋條件」的「交集」結果 |

---

## 使用情境

> 查詢公司設計驗證部所有人的分機表

```
phone 設計驗證部
```

查詢公司內有關 name 的分機表

```
phone name
```

查詢公司設計驗證部 name 的分機表

```
phone 設計驗證部 name
```

查詢公司分機號碼的對應人

```
phone 5711
```

查詢公司米勒會議室的分機

```
phone 米勒
```
