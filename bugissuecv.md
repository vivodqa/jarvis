# 指令名稱：bugissuecv

| 說明 | 條件數 | 來源 | 行為 |
| -| - | - | - |
|  在韌體驗證組中找尋符合條件的bug issue  | 3 | DQA redmine |針對「subject, description, id, status, tracker, custom fields, project目錄」欄位找尋，符合所有「搜尋條件」的「交集」之 韌體驗證組bug|

***
## 使用情境 
>查詢韌體驗證組發過有關「VAST」的 bug

```
bugissuecv VAST
```
>查詢韌體驗證組發過有關「延遲、lag」的 bug

```
bugissuecv 延遲 lag
```
